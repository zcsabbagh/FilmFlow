# Visualization Inspiration

Drop screenshots and images of data visualizations you want FilmFlow to emulate.
The agent will reference these when designing scenes.

Organize by type:
- `bar-charts/` — bar chart styles
- `line-charts/` — line and area charts
- `maps/` — choropleth, dot maps
- `stat-cards/` — big number displays
- `timelines/` — chronological layouts
- `comparisons/` — A vs B, dot grids
- `misc/` — other interesting styles
