# FilmFlow Quality Rubric — Comparative Against Real Vox

This rubric is used when evaluating FilmFlow output. Scores are relative to
actual Vox video production quality, NOT general video quality.

**10 = indistinguishable from real Vox. 5 = clearly amateur. 1 = static slides.**

## Categories

### Motion Design (weight: 25%)
- 9-10: Multi-layered animations, objects deform and interact, physically-driven motion
- 7-8: Creative transitions, varied animation types, good easing
- 5-6: Smooth basic animations (fades, slides, spring), but limited variety
- 3-4: Simple linear reveals, basic bar/line drawing
- 1-2: Static or jerky, no animation

### Visual Density (weight: 25%)
- 9-10: Every frame packed with information — imagery, diagrams, icons, annotations, visual metaphors
- 7-8: Background imagery, supporting graphics, multiple layers per frame
- 5-6: Charts with labels and sources, some supporting elements
- 3-4: Charts on blank backgrounds, minimal annotation
- 1-2: Text-only slides

### Color & Typography (weight: 15%)
- 9-10: Brand-level consistency, contextual color variation, perfect hierarchy
- 7-8: Good palette, clear hierarchy, custom fonts loading correctly
- 5-6: Decent colors, basic hierarchy, some font issues
- 3-4: Flat palette, poor hierarchy, system fonts
- 1-2: No visual design consideration

### Narrative Rhythm (weight: 20%)
- 9-10: Visuals deeply explain concepts as they're narrated, perfect sync, each frame teaches
- 7-8: Key beats hit on time, visuals illustrate and enhance narration
- 5-6: General timing ok, visuals match topic but don't enhance understanding
- 3-4: Visuals loosely related to narration, poor timing
- 1-2: No sync between audio and visuals

### Production Polish (weight: 15%)
- 9-10: Seamless transitions, consistent branding, source citations, end screen
- 7-8: Clean transitions, consistent style, proper attribution
- 5-6: Basic fades, mostly consistent, some rough edges
- 3-4: Jarring transitions, inconsistent styles, missing citations
- 1-2: No transitions, broken layout

## Baseline Scores (v7, April 2026)
- Motion Design: 3/10
- Visual Density: 2/10
- Color & Typography: 5/10
- Narrative Rhythm: 4/10
- Production Polish: 3/10
- **Overall: 3.3/10**
